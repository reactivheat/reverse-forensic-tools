# Roadmap v1

## Overview
Roadmap targets engineering maturity and a stable platform contract.

## Phase 0: CI & Security Baseline (1–2 weeks)
- Make CI deterministic across target platforms.
- Remove portability-breaking dev dependencies.
- Implement Guardrails, PathPolicy, Redaction baseline.

## Phase 1: Platform Core (2–3 weeks)
- Plugin registry + command contract.
- Evidence and report schema.
- Versioned reporters.

## Phase 2: DFIR MVP Plugins (3–6 weeks)
- Disk timeline MVP (placeholder acceptable but runnable).
- Memory process/network MVP.
- Network PCAP sessionization IOC MVP.
- Logs anomaly MVP.

## Phase 3: Reporting & Hardening (2–4 weeks)
- Evidence manifests and integrity.
- Golden snapshot regression tests.

## Phase 4: Release Readiness (2–4 weeks)
- SBOM + dependency scanning.
- CodeQL + release automation.
- Governance docs.
